# CodeAlpha Hangman Game

This is a simple text-based Hangman Game built in Python as part of the CodeAlpha Python Programming Internship.

## Features
- Uses 5 predefined words
- Allows the player to guess one letter at a time
- Maximum 6 incorrect guesses
- Console-based gameplay
- Input validation for better user experience

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code
2. Open the terminal
3. Run the following command:

```bash
python hangman_game.py
```

## Project Objective
The goal of this project is to create a basic Hangman game using Python fundamentals such as:
- random module
- loops
- if-else statements
- strings
- lists
