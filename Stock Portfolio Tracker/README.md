# CodeAlpha Stock Portfolio Tracker

This is a simple Stock Portfolio Tracker built in Python as part of the CodeAlpha Python Programming Internship.

## Features
- Hardcoded dictionary of stock prices (e.g., AAPL, TSLA, GOOGL, AMZN, MSFT)
- User inputs stock symbols and quantities
- Calculates total value for each stock
- Computes total investment value for the whole portfolio
- Optionally saves the summary to a `portfolio_summary.txt` file

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code
2. Open the terminal
3. Run:

```bash
python stock_portfolio_tracker.py
```

4. Enter stock symbols and quantities when prompted
5. Type `done` when you finish adding stocks

## Project Objective
The goal of this project is to practice:
- Python dictionaries
- User input and validation
- Basic arithmetic operations
- Simple file handling
