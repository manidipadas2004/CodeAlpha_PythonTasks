# CodeAlpha Basic Chatbot

This is a simple rule-based chatbot built in Python as part of the CodeAlpha Python Programming Internship.

## Features
- Takes user input from the console
- Responds with predefined replies
- Handles basic messages like:
  - hello
  - how are you
  - what is your name
  - bye
- Uses simple Python logic with if-elif conditions
- Runs in a loop until the user types `bye`

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code
2. Open the terminal
3. Run the following command:

```bash
python basic_chatbot.py
```

4. Type your message in the terminal
5. Type `bye` to end the chat

## Project Objective
The goal of this project is to create a simple chatbot using Python fundamentals such as:
- functions
- loops
- input/output
- if-elif statements
